# Chromaterra — Case Study

A case study page for [Chromaterra](https://chromaterra.vercel.app), a personal React web app that extracts color palettes from travel photos and organizes them into a personal atlas.

---

## About

This page documents the origin, technical decisions, and feature set behind Chromaterra — a tool built to bridge hands-on color work (quilling, visual art) with development. The idea started with wanting real palettes from real places: São Miguel island, Bywater in New Orleans, Leimert Park in Los Angeles.

The case study covers:
- Why the project exists and where the idea came from
- A security migration from client-side API calls to a Vercel serverless proxy
- Component reorganization and CSS Modules migration
- Firebase auth cross-browser fix (signInWithRedirect → signInWithPopup)
- WCAG 2AA accessibility audit via Pa11y (zero errors)
- A screen gallery of the live React build with lightbox
